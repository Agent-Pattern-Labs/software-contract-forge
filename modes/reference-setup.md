# Setup Reference

Consumer projects should contain:

- `config/client-profile.yml`
- `config/sources.yml`
- `data/pipeline.md`
- `data/applications/`
- `reports/`
- `batch/batch-input.tsv`

Run:

```sh
create-software-contract-forge my-contract-pipeline
cd my-contract-pipeline
npm install
```

Then fill in the profile and sources before scan, qualify, apply, or proposal work.

## Client Profile Minimums

- Work types and excluded work.
- Technical strengths.
- Geography and timezone constraints.
- Rate bands and minimum commercial terms.
- Auto-submit settings.
- Required user-review triggers.
- Reusable proof files or case studies.
