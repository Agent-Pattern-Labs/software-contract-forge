# Tracker Mode

Report current contract pursuit state.

## Inputs

- `data/pipeline.md`
- `data/applications/`
- `reports/`
- `batch/tracker-additions/`

## Procedure

1. Summarize active opportunities by state.
2. List blocked items and the missing input needed.
3. List follow-ups due today or overdue.
4. List recent wins, losses, submissions, and drafts.

## Output

Keep output compact and action-oriented. Include file paths when pointing at records.
